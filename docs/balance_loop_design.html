平衡环设计：先做 PD 内环，再扩展 PID/速度外环

你现在已经能从 IMU 得到 pitch 和 yaw。下一步不要先追求完整 PID， 而是先建立“姿态符号确认 + 安全限幅 + 最小 PD 平衡内环”的闭环。 平衡车的核心是 pitch 内环，yaw 暂时只用于观察，后续做转向外环时再使用。

先做什么

现在应该做的第一件事不是“完整 PID 控制器”，而是把平衡内环的闭环骨架建起来。 对两轮平衡车来说，第一版应该使用 PD 控制，也就是只使用 pitch 误差和 pitch 角速度。

第 1 步：确认符号

确认车身前倾时 pitch 是正还是负，确认左右轮同向命令 [w, w] 让车向前还是向后。

第 2 步：加安全层

加入 IMU 超时、跌倒角度、轮速限幅、输出斜率限制。没有安全层时不要打开闭环。

限幅 急停

第 3 步：做 PD 内环

先只让车“扶住自己”。此阶段不用 yaw，也不用位置积分。

pitch gyro_y

第 4 步：再扩展外环

内环稳定后，再加入轮速阻尼、位置外环、速度外环和 yaw 转向控制。

后续

平衡内环第一版不要使用 I 项。I 项会在小车倒下、轮子限幅或传感器异常时积累误差， 很容易让轮子突然打满。等 PD 能站住以后，再考虑在速度外环或位置外环中使用很小的 I 项。

平衡环结构

平衡环是一个内环，它不直接关心小车要往哪里走，只关心车身是否接近竖直。 它订阅 IMU，计算 wheel velocity command，并发布到现有的 /wheel_velocity_controller/commands。

1. 读取 IMU
从 quaternion 得到 pitch，从 angular_velocity.y 得到 pitch_rate。
2. 计算姿态误差
pitch_error = pitch - pitch_target。初始目标 pitch_target = 0。
3. PD 控制
u = kp * pitch_error - kd * pitch_rate，再乘 output_sign 修正轮子方向。
4. 安全过滤
对 u 做限幅、斜率限制、跌倒保护、IMU 超时保护。
5. 发布左右轮速度
平衡内环阶段左右轮同向：[u, u]。

yaw 不是平衡内环的核心量。yaw 用于“朝向控制”和“转向控制”，后续可以通过 left = balance_u - turn_u、 right = balance_u + turn_u 混合进左右轮。

控制信号定义

变量 来源 含义 第一版用途
pitch IMU quaternion 车身绕 y 轴的前后倾角 主反馈量
pitch_rate imu.angular_velocity.y 车身前后倾倒角速度 D 项反馈，抑制震荡
yaw IMU quaternion 车身朝向角 先记录，不参与平衡内环
wheel_u 控制器输出 左右轮共同角速度命令，单位 rad/s 发布到轮速控制器
output_sign 参数 修正控制输出方向 如果轮子追反了，只改它 
pitch_error = pitch - pitch_target
raw_u = kp_pitch * pitch_error - kd_pitch * pitch_rate
wheel_u = output_sign * raw_u
wheel_u = clamp_and_rate_limit(wheel_u)
publish([wheel_u, wheel_u])

调参顺序

  1. 先在 imu_debug.world 中确认 pitch 符号、pitch_rate 符号、轮子命令方向。
  2. 设置 enabled: false，只看 /balance_controller/debug 输出是否合理。
  3. 设置很小的 max_wheel_velocity，例如 3.0 rad/s。
  4. 打开 enabled: true，先把 kp_pitch 从很小值开始加。
  5. 如果车身开始来回震荡，再增加 kd_pitch。
  6. 如果车向前倒时轮子向后跑，不要改 PID，只改 output_sign。
  7. PD 能短时间站住后，再加入轮速阻尼 kv_wheel 或速度外环。

不要同时调 kp_pitch、kd_pitch、output_sign 和限幅。 每次只改一个变量，否则很难判断问题来自符号、增益还是限幅。

PID 工具类

这里保留 PID 名字，但平衡内环第一版只用 P 和 D。D 项优先使用 IMU 陀螺仪测到的 pitch_rate，不要对 pitch 数值做差分，因为差分更容易放大噪声。 

#!/usr/bin/env python3

from dataclasses import dataclass
from typing import Optional


@dataclass
class PIDConfig:
    kp: float = 0.0
    ki: float = 0.0
    kd: float = 0.0
    integral_limit: Optional[float] = None
    output_limit: Optional[float] = None


class PID:
    def __init__(self, config: PIDConfig):
        self.config = config
        self.integral = 0.0
        self.previous_error = None

    def reset(self):
        self.integral = 0.0
        self.previous_error = None

    def compute(self, error, dt, measured_rate=None):
        if dt <= 0.0:
            return 0.0

        p_term = self.config.kp * error

        self.integral += error * dt
        if self.config.integral_limit is not None:
            limit = abs(self.config.integral_limit)
            self.integral = max(-limit, min(limit, self.integral))
        i_term = self.config.ki * self.integral

        if measured_rate is not None:
            d_error = -measured_rate
        elif self.previous_error is None:
            d_error = 0.0
        else:
            d_error = (error - self.previous_error) / dt
        d_term = self.config.kd * d_error

        self.previous_error = error
        output = p_term + i_term + d_term

        if self.config.output_limit is not None:
            limit = abs(self.config.output_limit)
            output = max(-limit, min(limit, output))

        return output

平衡控制节点

下面是第一版平衡内环节点。它不使用 yaw，不接 /cmd_vel， 只做 pitch 平衡。为了方便你当前的 ament_cmake 布局，这份代码把 PID 类也放在同一个文件里。 

#!/usr/bin/env python3

import math
from dataclasses import dataclass
from typing import Optional

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Imu
from std_msgs.msg import Float64MultiArray


def clamp(value, lower, upper):
    return max(lower, min(upper, value))


def quaternion_to_euler(x, y, z, w):
    sinr_cosp = 2.0 * (w * x + y * z)
    cosr_cosp = 1.0 - 2.0 * (x * x + y * y)
    roll = math.atan2(sinr_cosp, cosr_cosp)

    sinp = 2.0 * (w * y - z * x)
    if abs(sinp) >= 1.0:
        pitch = math.copysign(math.pi / 2.0, sinp)
    else:
        pitch = math.asin(sinp)

    siny_cosp = 2.0 * (w * z + x * y)
    cosy_cosp = 1.0 - 2.0 * (y * y + z * z)
    yaw = math.atan2(siny_cosp, cosy_cosp)

    return roll, pitch, yaw


@dataclass
class PIDConfig:
    kp: float = 0.0
    ki: float = 0.0
    kd: float = 0.0
    integral_limit: Optional[float] = None
    output_limit: Optional[float] = None


class PID:
    def __init__(self, config: PIDConfig):
        self.config = config
        self.integral = 0.0
        self.previous_error = None

    def reset(self):
        self.integral = 0.0
        self.previous_error = None

    def compute(self, error, dt, measured_rate=None):
        if dt <= 0.0:
            return 0.0

        p_term = self.config.kp * error
        self.integral += error * dt

        if self.config.integral_limit is not None:
            limit = abs(self.config.integral_limit)
            self.integral = clamp(self.integral, -limit, limit)
        i_term = self.config.ki * self.integral

        if measured_rate is not None:
            d_error = -measured_rate
        elif self.previous_error is None:
            d_error = 0.0
        else:
            d_error = (error - self.previous_error) / dt
        d_term = self.config.kd * d_error

        self.previous_error = error
        output = p_term + i_term + d_term

        if self.config.output_limit is not None:
            limit = abs(self.config.output_limit)
            output = clamp(output, -limit, limit)

        return output


class BalanceControllerNode(Node):
    def __init__(self):
        super().__init__("balance_controller_node")

        self.declare_parameter("enabled", False)
        self.declare_parameter("imu_topic", "/imu/data")
        self.declare_parameter("command_topic", "/wheel_velocity_controller/commands")
        self.declare_parameter("debug_topic", "/balance_controller/debug")
        self.declare_parameter("control_rate", 200.0)
        self.declare_parameter("pitch_target", 0.0)
        self.declare_parameter("kp_pitch", 2.0)
        self.declare_parameter("kd_pitch", 0.12)
        self.declare_parameter("output_sign", 1.0)
        self.declare_parameter("max_wheel_velocity", 3.0)
        self.declare_parameter("max_wheel_accel", 60.0)
        self.declare_parameter("fall_angle_rad", 0.61)
        self.declare_parameter("imu_timeout_sec", 0.05)

        self.enabled = bool(self.get_parameter("enabled").value)
        self.pitch_target = float(self.get_parameter("pitch_target").value)
        self.output_sign = float(self.get_parameter("output_sign").value)
        self.max_wheel_velocity = float(self.get_parameter("max_wheel_velocity").value)
        self.max_wheel_accel = float(self.get_parameter("max_wheel_accel").value)
        self.fall_angle_rad = float(self.get_parameter("fall_angle_rad").value)
        self.imu_timeout_sec = float(self.get_parameter("imu_timeout_sec").value)

        kp = float(self.get_parameter("kp_pitch").value)
        kd = float(self.get_parameter("kd_pitch").value)
        self.pid = PID(PIDConfig(
            kp=kp,
            ki=0.0,
            kd=kd,
            output_limit=self.max_wheel_velocity,
        ))

        imu_topic = self.get_parameter("imu_topic").value
        command_topic = self.get_parameter("command_topic").value
        debug_topic = self.get_parameter("debug_topic").value

        self.latest_pitch = None
        self.latest_pitch_rate = None
        self.latest_yaw = None
        self.last_imu_time = None
        self.last_control_time = self.get_clock().now()
        self.last_wheel_command = 0.0

        self.command_pub = self.create_publisher(Float64MultiArray, command_topic, 10)
        self.debug_pub = self.create_publisher(Float64MultiArray, debug_topic, 10)
        self.create_subscription(Imu, imu_topic, self.on_imu, 20)

        control_rate = float(self.get_parameter("control_rate").value)
        self.create_timer(1.0 / control_rate, self.on_timer)

        self.get_logger().warn(
            "Balance controller started. enabled={} kp={:.3f} kd={:.3f}".format(
                self.enabled,
                kp,
                kd,
            )
        )

    def on_imu(self, msg):
        q = msg.orientation
        _, pitch, yaw = quaternion_to_euler(q.x, q.y, q.z, q.w)
        self.latest_pitch = pitch
        self.latest_yaw = yaw
        self.latest_pitch_rate = msg.angular_velocity.y
        self.last_imu_time = self.get_clock().now()

    def rate_limit(self, target, dt):
        max_delta = self.max_wheel_accel * dt
        delta = clamp(target - self.last_wheel_command, -max_delta, max_delta)
        limited = self.last_wheel_command + delta
        self.last_wheel_command = limited
        return limited

    def publish_wheels(self, wheel_velocity):
        self.command_pub.publish(Float64MultiArray(data=[
            wheel_velocity,
            wheel_velocity,
        ]))

    def stop_wheels(self):
        self.last_wheel_command = 0.0
        self.pid.reset()
        self.publish_wheels(0.0)

    def on_timer(self):
        now = self.get_clock().now()
        dt = (now - self.last_control_time).nanoseconds / 1e9
        self.last_control_time = now

        if self.latest_pitch is None or self.last_imu_time is None:
            self.stop_wheels()
            return

        imu_age = (now - self.last_imu_time).nanoseconds / 1e9
        if imu_age > self.imu_timeout_sec:
            self.get_logger().warn("IMU timeout, stopping wheels", throttle_duration_sec=1.0)
            self.stop_wheels()
            return

        pitch = self.latest_pitch
        pitch_rate = self.latest_pitch_rate or 0.0
        pitch_error = pitch - self.pitch_target

        if abs(pitch) > self.fall_angle_rad:
            self.get_logger().warn(
                "Fall angle reached: pitch={:.2f} deg".format(math.degrees(pitch)),
                throttle_duration_sec=1.0,
            )
            self.stop_wheels()
            return

        raw_u = self.pid.compute(
            error=pitch_error,
            dt=dt,
            measured_rate=pitch_rate,
        )
        signed_u = self.output_sign * raw_u
        saturated_u = clamp(
            signed_u,
            -self.max_wheel_velocity,
            self.max_wheel_velocity,
        )
        wheel_u = self.rate_limit(saturated_u, dt)

        if self.enabled:
            self.publish_wheels(wheel_u)
        else:
            self.stop_wheels()
            wheel_u = 0.0

        self.debug_pub.publish(Float64MultiArray(data=[
            pitch,
            pitch_rate,
            pitch_error,
            raw_u,
            wheel_u,
            self.latest_yaw or 0.0,
            1.0 if self.enabled else 0.0,
        ]))


def main():
    rclpy.init()
    node = BalanceControllerNode()
    try:
        rclpy.spin(node)
    finally:
        node.stop_wheels()
        node.destroy_node()
        rclpy.shutdown()


if __name__ == "__main__":
    main()

    pitch_target: 0.0
    kp_pitch: 2.0

    max_wheel_accel: 60.0
    fall_angle_rad: 0.61
    imu_timeout_sec: 0.05
参数 建议初值 说明
enabled false 先只观察 debug，不马上让轮子动。
kp_pitch 1.0 ~ 3.0 先从小值开始加，直到轮子明显追车体。
kd_pitch 0.05 ~ 0.3 用于抑制来回震荡。
output_sign 1.0 或 -1.0 只用于修正轮子追车体方向。
max_wheel_velocity 3.0 rad/s 第一轮调试不要直接使用 URDF 的 35 rad/s 上限。

判断方向的标准很简单：车身向前倒时，轮子应该向前追；车身向后倒时，轮子应该向后追。 如果方向相反，只改 output_sign。
